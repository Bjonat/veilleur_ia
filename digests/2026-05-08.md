#### 1. Avancées Architecturales et Représentation des LLM

*   **Représentation des Tokens Améliorée ([1] TIDE)** : Une nouvelle approche de l'indexation des tokens dans les LLM est proposée pour résoudre les problèmes de tokens rares et de "collapse contextuel", suggérant que chaque couche devrait connaître le token sous le contexte.
*   **Compréhension du Contexte Longue ([15] MiA-Signature)** : Introduction du concept de "Mindscape Activation Signature" (MiA-Signature), une représentation compressée de l'activation globale pour une meilleure compréhension des contextes longs.
*   **Optimisation des Architectures MoE ([18] UniPool)** : Proposition d'un pool d'experts globalement partagé pour les architectures Mixture-of-Experts (MoE), remettant en question l'allocation rigide par couche pour une meilleure efficacité.
*   **Encodage des Rôles Sociaux ([26] The Granularity Axis)** : Démonstration que les LLM encodent la granularité des rôles sociaux (micro à macro) dans leurs représentations internes, ouvrant des pistes pour une meilleure interprétation.
*   **Contrôle du Raisonnement et de la Divulgation ([36] When to Think, When to Speak)** : Une méthode pour découpler le raisonnement interne des LLM de leur engagement public, permettant une délibération privée avant la divulgation pour des réponses plus précises.

#### 2. Optimisation de l'Apprentissage et Alignement des LLM

*   **Correction des Biais d'Agrégation en RL ([4] Balanced Aggregation)** : Identification et correction des biais d'agrégation dans les méthodes GRPO (Group Relative Policy Optimization) pour l'apprentissage par renforcement avec récompenses vérifiables (RLVR).
*   **Attribution de Crédit dans les Agents LLM ([10] A^2TGPO)** : Une nouvelle méthode d'optimisation de politique pour les agents LLM multi-tours, permettant une meilleure attribution de crédit aux actions individuelles.
*   **Amélioration du Raisonnement par Perturbation ([14] Nonsense Helps)** : Utilisation de la perturbation de l'espace de prompt pour surmonter le problème du "zéro-avantage" dans GRPO, améliorant ainsi les capacités de raisonnement des LLM.
*   **Évaluation du Raisonnement par RL ([23] Can RL Teach Long-Horizon Reasoning)** : Introduction de ScaleLogic, un cadre pour étudier comment l'apprentissage par renforcement (RL) peut améliorer le raisonnement à long terme des LLM, avec un contrôle indépendant sur la difficulté.
*   **Alignement Robuste en Environnements Non-Stationnaires ([29] Turning Drift into Constraint)** : Proposition d'Autonomous Preference Optimization (APO) pour aligner le raisonnement de MLLM en environnements changeants, traitant les divergences comme des contraintes dynamiques.
*   **RL Résiduel pour le Raisonnement ([38] ResRL)** : Un nouvel apprentissage par renforcement résiduel avec projection d'échantillons négatifs pour stimuler le raisonnement des LLM sans sacrifier la diversité de génération.

*   **Récupération pour Agents de Raisonnement ([47] Rethinking Reasoning-Intensive Retrieval)** : Une étude sur l'amélioration des systèmes de récupération pour les agents de recherche, en se concentrant sur la fourniture de preuves complémentaires pour le raisonnement.
